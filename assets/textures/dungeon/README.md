# Dungeon Textures

Drop PNG overrides here to replace procedural textures generated by TextureGenerator.js.
TextureGenerator performs a HEAD check on load — if the file exists it swaps to the PNG automatically.

## Spritesheet tiles (from dungeon_v2.png — 512×512 each)

|--------------------|---------------------------------|----------------------|
| cracked_floor.png  | Cracked stone slab floor        | `cracked_floor`      |

## Usage in levels / DungeonLoader

Level JSON `theme` field maps to texture names via GeometryFactory:
- `"theme": "crypt"` → uses `shadow_wall` + `cracked_floor`
- `"theme": "cave"`  → uses `rough_wall` + `dark_floor`
- `"theme": "void"`  → uses `stained_wall` + `arcane_floor`
- `"theme": "ruin"`  → uses `slate_wall` + `cracked_tile`

To use in DungeonLoader, pass theme wall/floor names to GeometryFactory:
```javascript
this.geometryFactory.getTexture(`${theme}_wall`)   // e.g. 'shadow_wall'
this.geometryFactory.getTexture(`${theme}_floor`)  // e.g. 'cracked_floor'
```
