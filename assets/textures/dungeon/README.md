# Dungeon Textures

Drop PNG overrides here to replace procedural textures generated by TextureGenerator.js.

| Filename          | Size    | Usage                        |
|-------------------|---------|------------------------------|
| stone_wall.png    | 512×512 | Wall blocks (tileable)       |
| stone_floor.png   | 512×512 | Floor slabs (tileable)       |
| ceiling_stone.png | 256×256 | Ceiling (tileable)           |
| wood_door.png     | 256×512 | Door panels                  |

## Free CC0 sources

- https://ambientcg.com  → search "Rock029", "Bricks054", "Concrete"
- https://polyhaven.com  → Textures section
- https://opengameart.org

All must be tileable (seamless). Recommend 512×512 for walls/floor, 256×256 for ceiling.

To use: place file, then update GeometryFactory.js `get('stoneWall')` call to load via
`THREE.TextureLoader` instead of TextureGenerator when the file exists. The folder
structure is ready — wiring is Phase 1b.
